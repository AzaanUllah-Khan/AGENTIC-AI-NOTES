`Class Overview`

This class is **fully dedicated to HTML and CSS** and serves as the **only focused frontend-design class in the entire course**. The goal is to build a **strong web foundation**, enabling students to understand how websites are structured, styled, and made responsive.

This class also aligns with the **FreeCodeCamp Web Development certification**, helping students complete it with confidence.

`Learning Objectives`

By the end of this class, students will be able to:

* Understand how the web works at a structural level
* Write clean, semantic HTML
* Style websites using modern CSS
* Build responsive layouts
* Read and understand real-world frontend code

`What is HTML?`

**HTML (HyperText Markup Language)** is the **structure of a webpage**.

It defines:

* Headings
* Paragraphs
* Images
* Links
* Forms
* Layout sections

Think of HTML as the **skeleton of a website**.

`What is CSS?`

**CSS (Cascading Style Sheets)** controls the **appearance and layout** of a webpage.

It defines:

* Colors
* Fonts
* Spacing
* Layout
* Responsiveness

Think of CSS as the **skin and styling of the skeleton**.

`Core CSS Topics Covered`
